Built With Excellence

Technology Stack

A fully Kotlin-first, Kotlin Multiplatform codebase powering both Android and iOS from a shared business layer.

UI Layer
Jetpack Compose SwiftUI MVI Architecture StateFlow / SharedFlow
KMM Business
Kotlin Multiplatform Kotlin Coroutines ViewModels (shared) Repositories Use Cases
Data Layer
Ktor HTTP Client kotlinx.serialization SQLDelight
Infrastructure
Koin DI (Multiplatform) Analytics Logger Tracker Feature Flags Deep Linking
🔵
Jetpack Compose
Android UI
Declarative UI toolkit for Android. All BENO Android screens are built entirely in Compose with a custom design system.
🍎
SwiftUI
iOS UI
Declarative iOS UI framework. The iOS app consumes KMM ViewModels directly via the XCFramework.
🌐
Ktor
HTTP Client · KMM
Asynchronous HTTP client shared across Android and iOS. Handles all REST API calls with interceptors and auth tokens.
📦
kotlinx.serialization
JSON Parsing · KMM
Official Kotlin serialization library. Used alongside Ktor for type-safe JSON parsing across all platforms.
🗄️
SQLDelight
Database · KMM
Type-safe SQL database with KMM support. Generates Kotlin APIs from .sq files, shared between Android and iOS.
⚡
Kotlin Coroutines
Async · KMM
Structured concurrency for asynchronous programming. Used everywhere for network calls, database IO, and UI state.
💉
Koin
Dependency Injection · KMM
Lightweight dependency injection framework with Kotlin Multiplatform support. Wires the entire app without code generation.
🎯
Kotlin (100%)
Language
The entire codebase — Android, iOS business logic, data, and infrastructure — is written in Kotlin using KMM.
